# Portfolio Website

A premium personal portfolio website for AI Data & QA services.

## Features
- Modern UI (glass + gradient)
- Project case studies with modal
- Contact form (Formspree)
- Fully GitHub Pages compatible

## Setup

1. Clone or download repo
2. Open index.html

## Deployment

1. Push to GitHub
2. Go to Settings → Pages
3. Select main branch
4. Your site goes live
